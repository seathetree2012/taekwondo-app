🥋 태권도 자세 코치

시작하기 전에 한 번만 확인해주세요

본 약관은 「태권도 자세 코치」(이하 "서비스") 이용에 관한 사항을 정합니다. 끝까지 읽어주신 후 아래 항목에 동의해주세요. 이해가 어려운 부분이 있으면 보호자·지도자와 함께 확인해주세요.

1. 서비스 소개

본 서비스는 사용자가 업로드한 태권도 동작 영상·사진을 AI(인공지능)가 분석하여 자세에 대한 텍스트 피드백을 제공하는 도구입니다. 본 서비스는 다음 항목을 제공하지 않습니다:

  • 점수·등급·순위·합격 여부
  • 다른 사용자와의 직접 비교
  • 심사·시험·승급 결과 보장
  • 의료·건강·재활 조언

2. 수집하는 정보

  • 이름 — 본인이 직접 입력 (실명 아니어도 됨, 별명 가능)
  • PIN 4자리 — 본인 기록 잠금용
  • 업로드한 영상·사진 — 분석 대상 자세
  • 분석 결과 — AI가 생성한 텍스트 피드백
  • 썸네일 — 일기·기록 화면용 작은 이미지

주민등록번호·전화번호·이메일·주소는 수집하지 않습니다.

3. 외부 전송 (중요)

분석을 위해 업로드한 영상·사진은 Google LLC가 운영하는 Gemini AI 서버로 전송됩니다. 해당 서버는 미국 등 해외에 위치할 수 있습니다. Google 정책상 업로드된 파일은 약 48시간 후 자동 삭제되며, Google은 이 데이터를 모델 학습 등 다른 용도로 사용하지 않는다고 명시하고 있습니다.

참고: Google Gemini API 데이터 사용 정책은 Google 측 정책 변경에 따라 달라질 수 있으며, 본 서비스는 이를 보증하지 않습니다.

4. 데이터 보관 위치

이름·PIN·분석 결과·썸네일은 본인 휴대폰 내 브라우저 저장소(localStorage)에만 저장됩니다. 본 서비스 운영자는 별도 서버에 사용자 정보를 저장하지 않으며, 이름·PIN·결과를 열람할 수 없습니다.

휴대폰을 변경하거나 브라우저 데이터를 삭제하면 모든 기록이 사라집니다. 백업 기능은 제공되지 않습니다.

5. AI 분석의 한계

본 서비스의 AI는 영상·사진 한 장면만으로 자세를 평가하므로 다음과 같은 한계가 있습니다:

  • 촬영 각도·조명·해상도에 따라 결과가 달라질 수 있음
  • 실제 지도자의 평가와 다를 수 있음
  • 같은 동작을 두 번 분석해도 결과가 미세하게 다를 수 있음
  • AI가 명백한 실수를 할 수 있음

AI 피드백은 참고용일 뿐이며, 실제 지도자·코치의 가르침을 대체할 수 없습니다.

6. 안전·부상 면책

본 서비스는 의료·건강·운동처방 서비스가 아닙니다. AI 피드백을 따라 동작을 시도하다 발생한 부상·통증·사고에 대해 본 서비스 운영자는 책임을 지지 않습니다. 통증이 있거나 컨디션이 좋지 않으면 무리하지 마시고, 새로운 동작은 반드시 자격 있는 지도자의 지도 아래 연습해주세요.

7. 만 14세 미만 사용자

한국 개인정보보호법상 만 14세 미만 아동의 개인정보 처리에는 법정대리인(보호자)의 동의가 필요합니다. 만 14세 미만인 경우, 회원가입 시 다음을 함께 진행해주세요:

  • 보호자가 본 동의서를 함께 읽기
  • 가입 화면에서 보호자 이름 입력
  • 보호자가 항상 옆에서 사용을 지켜보기

8. 광고·제3자 공유 없음

본 서비스는 광고를 표시하지 않으며, 사용자 데이터를 수익화하지 않습니다. Google Gemini API 호출에 필요한 영상·사진 전송 외에는 어떤 제3자에게도 데이터를 제공하지 않습니다.

9. 사용자 권리

  • 언제든 메뉴에서 본인 계정·기록 삭제 가능
  • 브라우저 설정에서 사이트 데이터 삭제 가능
  • 특정 분석 결과만 골라 삭제 가능
  • 서비스 이용을 언제든 중단 가능

10. 약관 변경

본 약관은 서비스 개선이나 법령 변경에 따라 수정될 수 있습니다. 중요한 변경이 있을 경우 다음 접속 시 새 동의 절차를 다시 안내합니다.

11. 문의

서비스 문의나 개인정보 관련 요청은 가입 시 안내된 연락처로 보내주세요. (개인 프로젝트 단계로, 답변까지 시간이 걸릴 수 있습니다.)

이 약관은 대한민국 법률을 따르며, 분쟁 발생 시 본 서비스 운영자의 주소지 관할 법원을 1심 법원으로 합니다.
